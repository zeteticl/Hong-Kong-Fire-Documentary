## 二、整合資源平台

- **功能**：集合物資、避難中心、心理支援等各類資源

---

### 2.3 宏福苑消息

- **網址**：<https://blaze-info.vercel.app/>
- **功能**：
  - 緊急消息更新
  - 避難中心資訊
  - 物資分類
  - 其他支援資訊

---

### 2.4 大埔火警危機支援 - 社區資源管理

- **網址**：<https://supporttaipohk.com/>
- **核心功能**：
  - **上傳資源**：市民可上報支援資訊，由AI初步整合
  - **義工定位**：使用GPS追蹤當值義工位置（無人力調度功能）
  - **消息可信度投票**：用戶可對信息真偽進行投票
  - **需求與提供區**：居民可留言回應，尋求或提供協助
- **特點**：社區驅動的資源管理，強調信息驗證

## 七、使用說明

### 如何快速找到所需資源？

| 需求 | 推薦平台 |
|------|--------|
| 尋找失聯親友 | 報平安APP、尋親名單、Notion資源表 |
| 查詢避難中心位置 | 物資地圖、大埔火災資源整合 |
| 獲取物資援助 | 大埔火災資源整合、互助站 |
| 報名做義工 | 宏福苑互助 (volunteer tasks)、社區資源管理 |
| 上報支援資訊 | 社區資源管理、大埔火災資源整合 |
| 跟進最新消息 | Telegram群組、宏福苑消息、各平台新聞 |

## 八、社群責任聲明

- **信息驗證**：使用多平台資訊時，建議交叉驗證，特別是涉及資源分佈的消息
- **隱私保護**：提交資訊時請確認平台隱私政策
- **持續更新**：本清單資料截至2025年11月28日，實際可用資源可能持續增加或調整

## 九、資訊來源與歸檔

本頁面資訊來自於社群自發開發的各類支援平台，均於火災發生後48小時內推出。平台多由技術志願者、設計師、社會企業等迅速開發，旨在協調社區力量。

    Video Collection Guide

    This guide explains how to submit video footage and metadata to this project.

    Video Naming Convention

    Video File Naming

    Recommended naming format:

    • Format: YYYY-MM-DD_platform_post-id_video-number.mp4
    • Example: 2025-11-26_threads_DRgr4zOgdp7_01.mp4

    Metadata File Naming

    Each video should have corresponding Chinese and English metadata files, using -zh and -en suffixes:

    • Chinese metadata format: YYYY-MM-DD_platform_post-id_video-number-zh.md
    • English metadata format: YYYY-MM-DD_platform_post-id_video-number-en.md
    • Examples:
    • 2025-11-26_threads_DRgr4zOgdp7_01-zh.md (Chinese)
    • 2025-11-26_threads_DRgr4zOgdp7_01-en.md (English)

    Metadata File Structure

    Each video has separate Chinese and English metadata files with the following sections:

    Basic Information

    • File Name
    • Title

    Source Information

    • Original URL
    • Source Platform
    • Publisher
    • Post Date

    Collection Information

    • Collection Time: See initial commit time (see git commit) - automatically retrieved from git commit history
    • Collection Method: Please refer to Collection Method options in main README
    • Collector: See initial committer (see git commit) - automatically retrieved from git commit history

    Video Description

    • Brief description of video content, angle, time point, or key scenes
    • If AI-generated, marked with "Generated by AI, pending supplement"

    Technical Information

    • File Hash (SHA-256)
    • Duration
    • Resolution
    • File Size

    Notes

    • Other relevant information, such as the video's position in the post, etc.

    Information Required When Submitting Videos

    To ensure videos can serve as valid evidence, each video should include the following complete information:

    Required Information

    1. Original URL

    • The original source URL of the video (e.g., Threads, Instagram, YouTube, etc.)
    • If multiple videos are from the same post, each video should include the complete post URL
    • If the source is no longer available, please note this and provide backup sources (e.g., Wayback Machine)

    2. Collection Time

    • Automatically retrieved from the initial git commit time when the file was first added
    • Format: See initial commit time (see git commit)

    3. Collection Method

    • Please refer to Collection Method options in main README

    4. Video Description

    • Required: Brief description of video content, angle, time point, or key scenes
    • Examples: "Ground-level shot showing fire spreading upward from lower floors", "Close-up of building exterior burning"
    • If AI-generated, mark with "Generated by AI, pending supplement"

    Optional Information

    5. File Hash (SHA-256) [Recommended]

    • If video file was downloaded, provide SHA-256 hash
    • Mac/Linux command: shasum -a 256 filename.mp4
    • Windows command: certutil -hashfile filename.mp4 SHA256

    6. Collector

    • Automatically retrieved from the initial git commit author when the file was first added
    • Format: See initial committer (see git commit)

    7. Technical Information

    • Video duration, resolution, file size, etc. (if known)

    8. Notes

    • Other relevant information, such as the video's position in the post, whether it's a repost, etc.

    Video Categorization

    Videos are organized using a multi-level categorization system:

    Primary: By Content Type

    • On-site Footage: Citizen-recorded videos from the scene
    • News Reports: Media coverage and news segments
    • Official Statements: Government, fire department official releases
    • Interviews: Interviews with witnesses, officials, experts
    • Analysis Videos: Technical analysis, expert commentary videos
    • Aftermath Documentation: Post-fire documentation videos

    Secondary: By Source Platform

    Within each content type, videos are further organized by source platform:

    • Social media platforms (Threads, Instagram, YouTube, Facebook, etc.)
    • News media websites (HK01, Now News, RTHK, etc.)
    • Official websites (Government News, Fire Services Department, etc.)

    Tertiary: By Date

    Finally, videos are sorted by date for chronological reference.

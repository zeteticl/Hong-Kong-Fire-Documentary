## 會議記錄

### 1) 審議及通過上次會議議程

#### 1.1

鄺主席向在場各位宣讀是晚會議議程。經審閱後，與會委員一致通過是次會議議程。

### 2) 審議及通過第十二屆管理委員會第三次全體委員會議記錄

#### 2.1

經審議後，與會委員一致通過第十二屆管理委員會第三次全體委員會議記錄。

### 3) 跟進第十二屆管理委員會第三次全體委員會議議決事項

#### 3.1

管理處總經理向與會委員匯報就第十二屆管理委員會第三次全體委員會會議的跟進議決事項如下：

#### 3.2

會議記錄第 3.4 項，有關集資戶口定期存款事宜。法團再於 2024 年 8 月 8 日 全體知悉。

---

第 1 頁，共 3 頁

    日定期一個月存款$16,000,000，年息為3.55%。

### 3.3

第6.3項，有關民意調查結果事項。有關結果已在2024年8月13日的第2期名稱通知內列載。

### 3.4

第9.1項，有關聘請大維修工程監督事宜。委員會以工程顧問公司提交的招標稿，一致同意應徵者須持有功能監督作業守則之適任技術人員T1-T3資歷及具有平安書。此外不少於五年處地盤監督負責持及維修駐場監督經驗。工作期如1年半定為合約期，另加6個月酬勞的合約。委員會建議進行面試及設立試用期。如甄選第5星後三星期仍未有消息，營立即開工程度的公司開會商議需否調整招標條件中的資歷要求安排。

### 3.5

第10項，有關清洗各座地下大堂及改施房冷氣機事宜。承辦商「寒宇」已安排於8月14日至16日進行上述清洗工作。

### 4) 跟進特別業主大會事宜

#### 4.1

鄰主席報告於上次大會時討論，已決定於2024年9月6日晚7點正，在中華基督教會銘賢紀念中學禮行特別業主大會，並在8月6日發出通告，將有關消息通知各全完居民。

#### 4.2

鄰主席表示及後再收到核亊通知，希望能以民政事務署名義去信核亊力申請場地，故此，已向大補民政事務處求取協助申請場地安排，以便業主大會可順利舉行。

#### 4.3

後設：2024年8月23日的上午在大補民政事務處及樓少學區議員的協助下，法團委員與校方代表召開聯合會議商討及籌算有關校方場地的安排，最終能於同日正式落出特別業主大會結額。鄰主席要求管理處帶校長方要求為民完討細購賣新三者有責任保險及盡快安排場地備足。視訊宣辨，工代人員及點票公司等的籌備工作。

### 5) 商討及繼續調整工程集賬事項事宜

#### 5.1

鄰主席表示截至2024年7月31日，已收到73%的業主管收了大維修的第7期集資費用，總集賑額約5千萬。因應第一期集資情況良好及參考了7月19日於國發出民意調查的結果，當中收到45%的回覆贊為現時合理，49.3%認為集賑階段摸下可再吸納性。經再計支付安排後，建議訂定6個月一期集資並須提交各業主聲冊黨款。右則設會議會致設抉釋工程項目的對務展品。

#### 5.2

經商議後，陞議決維主席動議凍止大維修工程賑資的最俊繼集日期如下：

|   |   |
|---|---|
| 第一期捐止日： | 2024年7月31日 |
| 第二期捐止日： | 2024年10月31日 |
| 第三期捐止日： | 2025年1月31日 |
| 第二期捐止日： | 2025年4月30日 |
| 第六期捐止日： | 2025年7月31日 |
| 第八期捐止日： | 2025年10月31日 |

#### 5.3

上述調集軍資期由林菁器及和議，委參具一致感坨通過。

#### 5.4

委具要求本建筑商竭配合調整後的集資期但不可影響建築施工速度。「宏業」後先表示不理解情況，惟法團必須細星處理及遊非因經營困難而不繳收的業主的款款，以必影響後繼建工程運作。

第2頁，共3頁

    5.5 委員表示對於未能依時供貨責的禁主，將按決約的規定處理，除非已向管理處或設備後工程部負已有請了居享安貴報示未能如期敞故項的禁主，法團將會透一了解情況及提供協助。如法團答委會未能協助，會鎮業主同意下轉介至區議員作進一步提供協助。

6) 其他事項
6.1 管委會會閣經工程顧問公司於2024年8月9日批准的第一期支付工程費用證明書，經工程顧問核查的已完成工程金額為$10,856,800，需保留10% 工程保固金$1,085,680，故第一期表款書金額總數為$9,771,120。總客開項目 後，洗濯費用車動數，陳凌答委員和熱支付上述款項。各委員一致贊成議過由法令維修專責工作組成員按付款機制發竑工程款項支票。

無其他事項，會議於晚上9時50分結束。

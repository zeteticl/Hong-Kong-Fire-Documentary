11.6 張司庫查詢如果業戶現時使用有線電視，現時保養商否會一起做維修？管理處回覆現有有線電視系統的業戶因接收電視信號屬不同系統，故業戶需安自行致電有線電視處理。如業戶以往常使用有線電視而該公司沒有供發服務時需單位接收天線接回大廈公共天線系統亦需客自行聯絡有線電視客服，解單位天線接回大廈公共天線系統。

11.7 經商議後，各委員一致通過取消大廈侍星衛星天線電視系統並由現時保養商收回大廈衛星系統裝置，而現時大廈公共天線系統保養服務則繼續進行。各委員簽署確認。
